# Reverse Shell

## Reverse shell for respective technologies

* PHP = php reverse shell
* IIS Web server = aspx, asp

## Methodology

* Step 1: Try to execute OS command first \(RCE\)
  * Linux
    * id
    * whoami
    * ifconfig
    * pwd
  * Windows
    * whoami
    * dir
    * ipconfig
* Step 2: Execute reverse shell
